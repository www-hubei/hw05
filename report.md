人工智能通识课 HW05 实验报告

1. 文章学习摘要
通过学习《计算机视觉》第10篇，我掌握了卷积神经网络（CNN）的基本组件：
卷积层 (Conv)：利用局部感受野提取边缘和纹理特征。
池化层 (Pooling)：通过降采样减小参数量，提高平移不变性。
全连接层 (FC)：将提取的抽象特征映射到 10 个分类结果。

 2. LeNet-5 模型结构说明
根据经典 LeNet-5 设计并适配 MNIST 数据集（$28 \times 28$），结构推导如下：

| 层级 | 类型 | 输出尺寸 | 核大小/步长 | 参数量 (约) |
| :--- | :--- | :--- | :--- | :--- |
| Input | 输入 | $1 \times 28 \times 28$ | - | 0 |
| Conv1 | 卷积 | $6 \times 28 \times 28$ | $5 \times 5$, Pad=2 | 156 |
| Pool1 | 池化 | $6 \times 14 \times 14$ | $2 \times 2$, Stride=2 | 0 |
| Conv2 | 卷积 | $16 \times 10 \times 10$ | $5 \times 5$, Stride=1 | 2,416 |
| Pool2 | 池化 | $16 \times 5 \times 5$ | $2 \times 2$, Stride=2 | 0 |
| FC1 | 全连接 | 120 | - | 48,120 |
| FC2 | 全连接 | 84 | - | 10,164 |
| Output | 全连接 | 10 | - | 850 |

3. 模型对比分析
基于同等训练条件（5 Epochs, Adam 优化器），对比结果如下：

| 指标 | 极简 CNN (SimpleCNN) | 经典 LeNet-5 |
| :--- | :--- | :--- |
| 测试集准确率 | 98.12%| 98.99% |
| 模型复杂度 | 较低 (1层卷积) | 较高 (2层卷积+多FC层) |

结论分析：
LeNet-5 的表现优于极简 CNN。主要原因是 LeNet-5 采用了更深层的卷积结构，能够学习到多层级的特征表示（从基础边缘到复杂形状），且全连接层的深度增加了模型的分类能力。
